# linux-homelab

A home lab environment designed for learning Linux system administration (RHEL/Rocky Linux) and preparing for the RHCSA exam.

## Technologies

- Rocky Linux 9
- VirtualBox
- VMware ESXi
- Podman
- Apache
- Nginx
- NTP
- Ansible
- LVM
- NFS
- SSH
- SELinux
- firewalld
- systemd

## Labs

- Apache
- NFS
- LVM
- Networking
- Containers
- Ansible

Each directory contains a description of the lab, the configuration files used, and configuration examples.
